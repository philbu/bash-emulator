# Description

This project does emulate a basic Bash shell with some commands.

# Working Commands

## File commands

* cat
* cd
* head
* ls
* mkdir
* pwd
* touch
* tail
* rm

## System commands

* !!
* clear
* date
* uname
* whoami (with description)

## Other commands

* echo
* help (not original)

# Added files

* /
* /home, /home/<user>, /home/<user>/(.bashrc|README.md|whoami.txt)
* /boot
